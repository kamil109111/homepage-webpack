Frontend

Co to jest frontend?

Frontend odpowiada za wygląd strony/aplikacji oraz interakcję z użytkownikiem. Jest generowany głównie przy uzyciu trzech technologii: HTML, CSS, JavaScript.

HTML

Co to jest HTML?

HTML - HyperText Markup Language, czyli hipertekstowy język znaczników. Opdowiada za terść strony, a właściwie za jego odpowiednie opakowanie. Składa się ze znaczników.

CSS

Co to jest CSS?

CSS - Cascading Style Sheets, czyli kaskadowe arkusze stylów. Odpowiada zaprezentację (wygląd strony).
